# GottaGoFinder
### (a.k.a the best finder out there)

## Requirements
- Node v14 at least
- npm v6.14 at least

#### Run it
You can run it using `npm install && npm start`

From here you can just type the path you want to visit, and either click `search` or press `enter`.
